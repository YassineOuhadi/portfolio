# **App Name**: Yassine Ouhadi Portfolio

## Core Features:

- Home Section: Display a brief introduction, professional title, and links to projects and contact information.
- About Me Section: Showcase a detailed overview of skills, experience, and focus areas.
- Projects Section: Display featured projects with descriptions, tech stacks, and links to source code or live demos.
- Contact Section: Provide a contact form and direct links to email, GitHub, and LinkedIn.
- Skill Extractor: Uses AI to analyze the projects section and extract all used skills

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) to convey professionalism and sophistication.
- Background color: Very light gray (#F0F0F0), almost white, for a clean and modern feel.
- Accent color: Electric purple (#BF00FF) to highlight key interactive elements.
- Body font: 'Inter', a sans-serif font, providing a modern, neutral, and readable style for all content.
- Headline font: 'Space Grotesk', a sans-serif font, to deliver computerized and techy feel to the headers. In the event the content is longer than short bursts, use 'Inter' as the body font in conjuction with Space Grotesk'
- Simple and professional icons to represent skills, technologies, and links.
- Clean and modern layout with a focus on readability and easy navigation. Utilize a grid system for consistent spacing and alignment.
- Subtle transitions and animations to enhance user experience and provide feedback on interactions.
